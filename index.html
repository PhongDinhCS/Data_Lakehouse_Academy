Data Lakehouse Engineering & Analytics

Detailed Training Syllabus & Roadmap (6 Months)

Duration: 24 Weeks
Sessions: 72 (3/Week)
Hands-on: 80%

Learning Objectives

Data Analyst Core Skills

  • Query & analyze data using Spark SQL on Databricks
  • Transform raw data into structured Silver/Gold datasets
  • Build automated dashboards & business insights
  • Apply Fact/Dimension modeling for Lakehouse

Data Engineer Core Skills

  • Design end-to-end Medallion Architecture pipelines
  • Implement Batch & Streaming (Kafka + Spark)
  • Orchestrate complex workflows using Apache Airflow
  • Performance tuning & Delta Lake optimization

PHASE 1: Data Analyst

Weeks 1 — 8

Fundamentals

Weeks 1-2

Databricks

Core Concepts

  • Data Lakehouse vs Warehouse Architecture
  • Medallion Design Pattern (Bronze/Silver/Gold)
  • Databricks Workspace, Clusters & Notebooks
  • DBFS & Cloud Object Storage Fundamentals

Hands-on Lab

"First Lakehouse Explorer"

Setting up Databricks Community, uploading raw CSV files to DBFS, and performing basic data reads using PySpark/SQL.

Advanced SQL

Weeks 3-4

Spark SQL

Core Concepts

  • CTEs & Complex Subqueries
  • Window Functions (RANK, LEAD, LAG)
  • Handling Nested Data (JSON/Arrays) in SQL
  • Optimizing query execution plans

Hands-on Lab

"Business Intelligence Queries"

Solving 10 real-world business cases: Revenue growth MoM, Top 5 customers per region, and churn indicators.

Data Transformation

Weeks 5-6

Delta Lake

Core Concepts

  • Bronze Layer: Schema-on-Read
  • Silver Layer: Cleansing, deduplication & NULL handling
  • Delta Lake basics: ACID transactions & Time Travel

Hands-on Lab

"Silver Pipeline Builder"

Writing SQL/PySpark notebooks to clean raw telematics data, enforcing schema, and storing as Delta tables.

Modeling & BI

Weeks 7-8

Modeling Power BI

Core Concepts

  • Star Schema Design in Lakehouse
  • Fact vs Dimension Tables
  • KPI Definition & Dashboard Storytelling

Final Project (P1)

"Lakehouse Analytics Project"

Full end-to-end notebook pipeline from Raw to a curated Gold layer with a Power BI dashboard.

PHASE 2: Data Engineer

Weeks 9 — 24

Month 3: PySpark & Medallion Deep Dive

PySpark Delta Lake
Topic: Scalable Transformations Sessions 25-30

Deep dive into Spark internals (Driver, Executor, Partitions). Master complex transformations using PySpark DataFrames API instead of SQL. Focus on handling schema evolution and data quality enforcement at the Silver layer.

Month 4: Real-time Streaming

Kafka Structured Streaming
Topic: Kafka Ingestion & Spark Streaming Sessions 37-48

Setting up Apache Kafka topics. Simulating real-time IoT data. Building Spark Structured Streaming jobs to ingest Kafka events into Delta Lake Bronze tables. Real-time window aggregations.

Month 5: Orchestration & DB Ops

Airflow PostgreSQL
Topic: Workflow DAGs & Ingestion Patterns Sessions 49-60

Learning Airflow operators. Scheduling pipelines to pull data from PostgreSQL (External) into Databricks. Handling failure retries and sensor-based triggers. Delta Lake performance tuning (Z-Order, Optimize).

Month 6: Capstone

Graduation

End-to-End Data Lakehouse System

Students will integrate all tools to build a multi-source pipeline (Kafka + SQL DB) into a unified Medallion Lakehouse, orchestrated by Airflow, with real-time monitoring dashboards.

Sessions 61-72 PORTFOLIO READY

Assessment & Teaching Structure

Teaching Style

Each session follows the 30/70 rule: 30% interactive lecture, 70% supervised coding labs.

Grading System

  • Weekly Assignments: 40%
  • Mini Projects: 20%
  • Final Capstone: 40%

Career Support

Resume review for DA/DE roles and interview simulation based on capstone architecture.
